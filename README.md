## Project Objectives
This project is a robust, blazingly-fast 🔥🔥🔥, Mandelbrot set generator and explorer. It implements:

- Efficient parallel computing
- A range of rendering options
- A lightweight, blisteringly-fast, and vigorous GUI 👐

## Running the App
First, download an appropriate distribution from releases

You can then either 
### Unzip the release(warning, it's big)
Open the folder in finder, and run ```./run-mandelbrot.sh```

### Leave the file zipped
run
```
cd /tmp && unzip <your/absolutePath/MandelbrotAppBundle-1.0.0.zip> -d MandelbrotApp && cd MandelbrotApp && ./run-mandelbrot.sh
```
